Testes de Chamados

No chamado, quero conseguir selecionar o sistema que esta com problema. Então deve-se ter a seguinte adição no sistema: Faça isso de forma coerente e desenvolva da melhor maneira possível. Deve ser possível, no grupo, adicionar os sistemas do grupo, para que posteriormente sejam selecionados no chamado, ESSA É UMA FUNÇÃO QUE BUSCA MELHORAR A IDENTIFICAÇÃO DO SISTEMA NO CHAMADO E o chamado como um todo.
